# AI Dev Workflow Template

Use this repository as a starting point for new projects.

## Create New Project From This Template

### One-time setup (on this repo)
1. Open this repository on GitHub.
2. Go to **Settings**.
3. Enable **Template repository**.

### Create a new project from the template
1. Open this repository on GitHub.
2. Click **Use this template**.
3. Choose owner, repository name, visibility, and create the repo.

## CLI Alternative

```bash
gh repo create my-new-project --template Jdodge1090/ai-dev-workflow-template --private --clone
```

Use `--public` instead of `--private` if needed.
